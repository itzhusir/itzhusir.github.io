###@RequestParam,@PathParm,@PathVariable和@@RequestBody

这四个注解都是用来传参数的，第一个是用来传递http://localhost:8080/page1?id=1 这种用的。
第二个和第三个用来处理http://localhost:8080/page1/1这种传参数的，后面这个是用来传对象用的。

###git 使用步骤
1.git clone url
2.git init
3.git branch name
4.git checkout name
5.git add .
6.git commit -am “desc”
7.git remote add name url 这表示与远程分支进行联系
~ git merge name 使name分支与master合并。解决冲突
8.git push –set-upstream name name 之后直接push就行
每次在git clone 之应该先git pull，同步git 的代码,在git push 之前要先git
